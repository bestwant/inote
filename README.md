# inote

记忆学习辅助工具

## Getting Started


### 预期实现

1、增加一条记忆
2、对记忆按照艾宾浩斯遗忘曲线提醒
3、可以查看进行中的，已完成的项目

### 已实现
1、增加一条记忆
2、可以查看进行中的，已完成的项目
3、可也手动强制完成进行中的，也可以把已完成的重新加入到进行中
